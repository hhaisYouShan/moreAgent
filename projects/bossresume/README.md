# BossResume migration reference

This directory contains material migrated from BossResume for compatibility extraction and historical comparison.

It is **not**:

- the current AI Software Company OS delivery goal;
- the canonical OS product or architecture source;
- the Stage 7 executable validation pack;
- a replacement for the BossResume repository.

Current purpose:

- preserve source material used to extract generic contracts and capabilities;
- support compatibility and parity analysis;
- identify project-specific assumptions that must remain in profiles or adapters.

The future BossResume real-project validation pack belongs under `validation/bossresume/` and becomes active only in roadmap Stage 7. BossResume business PRDs, business code, runtime state, product acceptance, and user acceptance remain in the BossResume repository.
